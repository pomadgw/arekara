# Arekara

This is a web app for counting events since it was created.

Track when the last time you ate that junk food or when the last time you stopped smoking! When you did again, you can reset the timer, and the app will record how many times you reset the timer and when you did it.

Inspired by [Time Since](https://play.google.com/store/apps/details?id=es.desaway.timesince&hl=in&gl=US).

Developed with TypeScript and Vue.
